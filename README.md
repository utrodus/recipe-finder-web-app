# Web App Cari Resep With Vue

Simple web App untuk mencari resep makanan dibuat dengan VueJs dan Bootstrap untuk submission Dicoding Kelas Front-end Developer

## Installation

How to Install :

```bash
npm install
```

## Usage

Start development :

```bash
npm run start-dev
```

How to build:

```bash
npm run build
```
